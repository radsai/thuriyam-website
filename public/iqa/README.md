# IQA Dashboard Screenshots

This directory should contain screenshots from the production IQA dashboard at https://dashboard.thuriyam.ai/

## Required Screenshots

Please add the following screenshots from the production dashboard:

1. **dashboard-overview.png** - Main dashboard overview showing all conversations, metrics, and insights
2. **conversation-analytics.png** - Detailed conversation analytics view with sentiment, compliance, and insights
3. **performance-metrics.png** - Performance metrics dashboard showing agent performance and quality scores
4. **sentiment-analysis.png** - Sentiment analysis dashboard with trends and insights
5. **compliance-monitoring.png** - Compliance monitoring dashboard with risk flagging
6. **agent-performance.png** - Individual agent performance metrics and rankings
7. **trends-analytics.png** - Historical trends and predictive analytics view
8. **filter-search.png** - Advanced filtering and search interface
9. **export-reports.png** - Export and reports functionality

## Instructions

1. Log into https://dashboard.thuriyam.ai/
2. Navigate to the IQA/Analytics section
3. Take screenshots of the various dashboard views listed above
4. Save them with the exact filenames listed above
5. Place them in this directory (`/public/iqa/`)

The page will automatically display these screenshots once they are added. If a screenshot is missing, a placeholder will be shown.
